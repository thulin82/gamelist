Small express api to use with howlongtobeat npm module


Setup
---------------------
### Get project
```
$ git clone git@github.com:thulin82/gamelist.git
```
### Get dependencies
```
$ npm install
```
### Run app
```
$ npm start
```
### Test app
```
$ npm test
```
